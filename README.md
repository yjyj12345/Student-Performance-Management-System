学生信息	
所有学生：学号、姓名、性别、年龄、班级
小学生：英语、数学、语文成绩 中学生：地理、历史成绩
大学生：专业课、英语、程序设计、高等数学成绩
添加功能：添加不同学生记录
查询功能：通过学号或姓名查询已添加的学生记录
显示功能：显示当前系统中所有学生记录
编辑功能：根据查询结果对相应的学生记录进行修改
删除功能：对已添加的学生记录进行删除
统计功能：参数统计，如学生人数、总分、单科平均分等
保存功能：将当前系统中各类学生记录存入文件
读取功能：将保存在文件中的信息读入系统以供使用
排序功能：按总分或单科成绩排名

void draw()：绘制注册登录界面，包括标题、输入框、按钮等。
void getuserdata()：获取输入的用户名和密码，循环接收字符，将当前输入的字符添加到字符串末尾，从而得到完整的用户名和密码，且输入过程中支持退格键删除已输入的字符操作。
void saveuserdata()：将输入的用户名和密码信息保存到用户信息文件中。若文件成功打开则把用户名和密码按照指定格式写入文件，并在提示注册成功；若文件无法打开则提示错误，并退出程序。
int verifyuserdata()：读取存储用户信息的文件，将文件中的用户名密码与当前输入的用户名密码进行对比验证，若对比验证成功则返回1，表示登陆成功；若对比验证失败则返回0，表示输入的用户名密码不正确，登录失败。
void buttonclick()：根据鼠标左键点击的位置判断是否点击了按钮，点击了注册按钮还是登录按钮，然后执行对应的操作。
查询学生void searchstudent()：提示用户输入要查询学生的姓名或学号，并接收输入内容存到query 中。初始化found 为 0，用于标记是否找到对应学生。然后通过for循环遍历students容器的所有学生对象，在循环中判断每个学生对象的姓名或者学号是否与输入的query相等，若相等则调用该学生对象的show函数展示其信息，并将 found 置为 1。最后根据found的值判断，如果仍为 0，意味着未找到匹配的学生记录，输出 “未找到相应记录！” 提示。
修改学生void editstudent()：同理，若找到了匹配的学生对象，就调用该学生对象的input函数，让用户重新输入该学生的相关信息以完成编辑操作

使用sort函数对students容器中的学生对象指针进行排序操作。通过 dynamic_cast 分别对传入的两个学生对象指针 a 和 b 进行动态类型转换判断，若 a 和 b 都能成功转换为相同学生的类型指针，意味着是该类学生对象。接着根据传入的学科名称（通过参数 subject 传递）来比较成绩，返回成绩高的学生对象排在前面的比较结果，让 sort 函数按此规则调整排序顺序。如果 a 和 b 不属于同一种学生类型或者其他不符合条件的情况，统一返回false，此时不改变原有的相对顺序。完成 sort 操作后，调用 printsortedbysubject将已经按照指定学科成绩排好的学生信息输出。
